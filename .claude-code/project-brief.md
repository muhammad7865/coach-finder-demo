# Coach Finder Project - Context

## Client Requirements
- Hosiery export business owner (Pakistan/USA)
- Needs to find coaches from small sports clubs (not big leagues)
- Target: LinkedIn, Facebook, Instagram scraping
- Output: Excel export with coach details

## What We're Building (MVP Demo)
1. Instagram scraper for small sports clubs
2. Search by: hashtag, location, follower count
3. Export to Excel: Name, Bio, Email, Follower count
4. Professional web UI

## Tech Stack Decided
- Frontend: Next.js 14 + TypeScript + Tailwind
- UI: shadcn/ui components
- Scraping: Apify SDK (Instagram Profile Scraper)
- Export: xlsx library
- Deployment: Vercel
